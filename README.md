# goodera-hackathon
## Prerequisites-
Make sure you have node & npm install in your system. 

### Clone the App

```
git clone <url>

cd goodera-hackathon
```
### To Start Server

open terminal in root ,then
```
cd server

npm install

npm run start
```

### To Start app

open another terminal in root ,then
```
cd client

npm install

npm run start
```
